# Claude Code — Operating Cadence

A self-contained reference for when and how to trigger Claude Code skills, with a
session-triage tool. Open `index.html` (or the Pages link) at the start or end of
a session to decide what to run.

- **Passive** — always on via CLAUDE.md
- **Ready** — invoke now
- **Enable** — register the skill first
- **Build** — small glue not yet written

Edit `index.html` and push to update.
